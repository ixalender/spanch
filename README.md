# spanch

```
NAME:
   spanch - Calls command when filesystem at specified path has been changed

USAGE:
   spanch help | -p . "system command with args"

VERSION:
   1.0

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --path value, -p value  filesystem path to watch (default: ".")
   --help, -h              show help (default: false)
   --version, -v           print the version (default: false)
```
